# CatchU
CatchU는 채용 사이트를 만들어보는 개인 프로젝트입니다.

사용 기술
- React(TypeScript)
- Express

* git clone
```
$ git clone https://github.com/gee05053/CatchU.git
```

* Run Server
```
$ cd CatchU
$ cd catchu-server
$ npm i
$ npm start
```

* Run Client (open another terminal)
```
$ cd CatchU
$ cd catchu-webui
$ npm i
$ npm start
```
